# CODSOFT Data Science Internship

## Overview

This repository contains the Data Science projects completed as part of the CODSOFT Data Science Internship Program. The projects demonstrate the application of machine learning techniques for classification and prediction tasks using Python and Scikit-learn.

## Projects Included

### 1. Iris Flower Classification

* Developed a machine learning model to classify Iris flowers into different species based on sepal and petal measurements.
* Implemented using a Decision Tree Classifier.
* Achieved high classification accuracy on the test dataset.

### 2. Titanic Survival Prediction

* Built a predictive model to determine passenger survival on the Titanic.
* Performed data preprocessing, feature selection, and model training.
* Implemented using Logistic Regression.

### 3. Sales Prediction Using Python

* Developed a regression model to predict sales based on advertising expenditure.
* Applied data analysis and machine learning techniques.
* Implemented using Linear Regression.
